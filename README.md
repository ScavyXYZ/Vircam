# Vircam - open source virtual camera project
Vircam can play photos or videos as if it were your webcam.

</video>
## Installation
1. Download latest release
2. Open terminal in the project root directory
3. Run one of these commands
  - if you have a good or average computer: `./RegisterDriver 60`
  - Very bad computer: `./RegisterDriver 30`
  - Potato: `./RegisterDriver 20`  

If you have difficulty installing it, you can watch the [video](https://github.com/ScavyXYZ/Vircam/raw/refs/heads/main/materials/tutorial.mp4) tutorial.

Nothing complicated
1. Select the *Vircam* playback device in the target programme.
